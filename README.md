# Machine-Learning-and-Deep-Learning-Labs.




Welcome to the  Machine Learning and Deep Learning Lab Repository**.  
This repository contains lab tasks, experiments, and implementations completed during the Machine Learning and Deep Learning laboratory course.

The course focuses on practical implementation of Machine Learning (ML) and Deep Learning (DL) concepts using Python and modern data science libraries.

---

# 📚 Course Overview

This repository covers:

- Data preprocessing and cleaning
- Feature engineering
- Data visualization
- Regression techniques
- Gradient descent optimization
- Overfitting and underfitting analysis
- Image dataset preparation
- Convolutional Neural Networks (CNNs)
- Data augmentation
- Multi-Layer Perceptrons (MLP)
- 1D CNNs
- Long Short-Term Memory Networks (LSTM)

---

# 🛠 Technologies & Libraries Used

- Python

- Matplotlib
- Scikit-learn
- TensorFlow / Keras
